# Temporary User Setup with Expiry Date

As part of the temporary assignment to the Nautilus project, a developer named `amer` requires access for a limited duration. To ensure smooth access management, create a temporary user account with an expiry date.

> Create a user named `amer` on **App Server 1** (Stratos Datacenter). Set the expiry date to `2024-01-28`, ensuring the username is lowercase.

## Steps

1. Connect to the server and run the following command to create the user with an expiration date:

```
    sudo useradd -m -e 2024-01-28 amer
    ```

2. Verify:

    ```sh
    cat /etc/passwd | grep amer
    sudo su amer
    ```
